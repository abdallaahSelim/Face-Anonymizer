# Face Anonymizer

A project to anonymize faces by applying a blur effect using the OpenCV library.

## Installation
```bash
pip install opencv-python numpy
```

## Running the Project
1. Open `main.py`.
2. Select the `mode` (image / video / webcam) on line 5.
3. Run the project:
```bash
python main.py
```

## Controls (Webcam Mode)
* **S**: Save a snapshot.
* **R**: Start/Stop recording video.
* **Q**: Close the program.

## Project Structure
* `main.py`: The main source code.
* `output/`: Folder containing the processed results.
* `requirements.txt`: List of required libraries.
